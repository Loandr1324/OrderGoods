1. Формируем отчет Остатки(Авотомат) в трёх базах 1С.
2. Сохраняем их в Папку "Исходные данные" с разным наименованием, в котором обязательно должно быть слово "Остатки".
3. Формируем Отчет товароведа в базе ТС и сохраняем в папку "Исходные данные" в файл, 
в наименовании которого обязательно должно быть слово "МО".
4. В отчете товароведа в настройках:
4.1. Убираем галочки со всех настроек, особенно с настройки "Убрать пустые строки".
4.2. Выбираем любой перид, главно чтобы конечная дата совпадала с текущим днём.
4.3. Выбираем те же папки номерклатуры, что и в отчете по "Остаткам" из п.2
5. Запускаем программу "Расчёт потребностей.exe"
6. Результатом выполнения программы будет создание двух файлов:
6.1. "Анализ потребностей.xlsx" - В этом файле отображаются суммированные остатки по всей компании, 
МО по всей компании, и в последней колонке проставлено количество требуемого товара к заказу.
Этот файл создаётся для того, чтобы вручную изменить автоматически посчитанные потребности 
и сформировать файл для отправки поставщику.
6.2. "Потребность для заказа.xlsx" - в этом файле удаляются все строки в которых нет потребностей 
и остаются только те позиции, потребность в котрых больше нуля.
